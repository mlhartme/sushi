Interface Root<T extends Node>

  • All Known Implementing Classes:
    ConsoleFilesystem, FileRoot, MemoryRoot, SshRoot, SvnRoot, TimeMachineRoot, WebdavRoot, ZipRoot

    public interface Root<T extends Node>
    A file system root like the drive letter on Windows or the Host Name for HttpFilesystem. The root is not a node. A root node is a node with an empty path.
