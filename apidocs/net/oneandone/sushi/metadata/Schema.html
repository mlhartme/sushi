Class Schema

  • java.lang.Object
    • net.oneandone.sushi.metadata.Schema
  • Direct Known Subclasses:
    AnnotationSchema, ReflectSchema

    public class Schema
extends Object
    A set of Types. Initially, the set consists of simple types only. Complex types can be created explicitly by invoking the add method or implicitly by overriding the complex method. Thus, metadata can be used as a factory for complex types.
